Diversity settings for the resistive sensing principle.
Applications » tadherence: Therapy Adherence Demo Application » Therapy Component » Sensing Component » Specific part for the Sensing Component

Detailed Description

This part defines possible diversity for the resistive sensing principle.

With this sensing method there are 2 possible principles as defined by SENSE_RES_SENS_PRINC_T. For both these principles, the pill vector is deduced from the admittance of the network/ladder. To determine the admittance, both the voltage and the current are measured. The following procedure is followed:

  1. The drive pin is driven by the DAC.
  2. Using the ADC the voltage of both the drive and the sense pin is measured.
  3. These values are combined to calculate the voltage drop caused by the resistive network/ladder.
  4. Using the I2D the current is measured (flowing into the sense pin), this current is equal to the current flowing through the network/ladder.
  5. The admittance of the network is then equal to the current divided by the voltage drop.
